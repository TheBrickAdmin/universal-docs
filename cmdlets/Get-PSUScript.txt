
NAME
    Get-PSUScript
    
SYNOPSIS
    Returns scripts defined within PSU.
    
    
SYNTAX
    Get-PSUScript [-Folder] <Folder> [-AppToken <String>] [-ComputerName <String>] [-Integrated] [-UseDefaultCredentials] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    Get-PSUScript [-Id] <Int64> [-AppToken <String>] [-ComputerName <String>] [-Integrated] [-UseDefaultCredentials] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    Get-PSUScript [-Identity] <Identity> [-AppToken <String>] [-ComputerName <String>] [-Integrated] [-UseDefaultCredentials] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    Get-PSUScript [-Name] <String> [-AppToken <String>] [-ComputerName <String>] [-Integrated] [-UseDefaultCredentials] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    Get-PSUScript [-AppToken <String>] [-ComputerName <String>] [-Integrated] [-Tag <Tag>] [-UseDefaultCredentials] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    
DESCRIPTION
    Returns scripts defined within PSU.

NOTES
    
    
        
    
    -------------------------- Example 1 --------------------------
    
    PS C:\> Get-UAScript
    
    Returns all scripts defined within PSU.
    -------------------------- Example 2 --------------------------
    
    PS C:\> Get-UAScript -Id 12
    
    Returns the script with ID 12.
    -------------------------- Example 3 --------------------------
    
    PS C:\> $Identity = Get-UAIdentity -Name 'Adam'
    PS C:\> Get-UAScript -Identity $Identity
    
    Returns all scripts created or modified by the identity 'Adam'.
    -------------------------- Example 4 --------------------------
    
    PS C:\> Get-UAScript -FolderId 12
    
    Returns all scripts with the folder with ID 12.
    -------------------------- Example 4 --------------------------
    
    PS C:\> Get-UAScript -Name 'Script1.ps1'
    
    Returns script 'Script1.ps1'
    -------------------------- Example 5 --------------------------
    
    PS C:\> $Tag = Get-UATag -Name 'Release'
    PS C:\> Get-UAScript -Tag $Tag
    
    Returns all scripts with the 'Release' tag.
