
NAME
    Get-PSUJobFeedback
    
SYNOPSIS
    Returns feedback requested by jobs.
    
    
SYNTAX
    Get-PSUJobFeedback [-AppToken <String>] [-ComputerName <String>] [-Integrated] -Job <Job> [-UseDefaultCredentials] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    Get-PSUJobFeedback [-AppToken <String>] [-ComputerName <String>] [-Integrated] -JobId <Int64> [-UseDefaultCredentials] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    Get-PSUJobFeedback [-AppToken <String>] [-ComputerName <String>] [-Integrated] [-UseDefaultCredentials] -RunId <Guid> [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    
DESCRIPTION
    Returns feedback requested by jobs.

        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false

        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Integrated [<SwitchParameter>]

        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Job <Job>
        The job to return feedback for.
        
        Required?                    true
        Position?                    named
        Default value                None
        Accept pipeline input?       True (ByValue)
        Accept wildcard characters?  false
        
    -JobId <Int64>
        The job ID to return feedback for.
        
        Required?                    true
        Position?                    named

        Accept pipeline input?       False

    -UseDefaultCredentials [<SwitchParameter>]
        Use default credentials when connecting to the management API

        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -RunId <Guid>

        Required?                    true

        Accept wildcard characters?  false
