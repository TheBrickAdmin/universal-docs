
NAME
    Get-PSUFolder
    
SYNOPSIS
    Returns folders defined within PSU.
    
    
SYNTAX
    Get-PSUFolder [-AppToken <String>] [-ComputerName <String>] [-Id <Int64>] [-Integrated] [-UseDefaultCredentials] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    Get-PSUFolder [-AppToken <String>] [-ComputerName <String>] [-Integrated] [-Name <String>] [-UseDefaultCredentials] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    Get-PSUFolder [-AppToken <String>] [-ComputerName <String>] [-Integrated] [-Parent <Folder>] [-UseDefaultCredentials] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    
DESCRIPTION
    Returns folders defined within PSU. Folders will be based on the internal and\or remote git repo.

        Position?                    named

        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Id <Int64>

        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Integrated [<SwitchParameter>]

        Required?                    false

        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Name <String>
        The name of the folder to return.

        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Parent <Folder>
        The parent folder to return children for.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False

    None

NOTES
    
    
        
    
    -------------------------- Example 1 --------------------------
    
    PS C:\> Get-UAFolder
    
    Returns all folders within PSU.
    -------------------------- Example 2 --------------------------
    
    PS C:\> Get-UAFolder -Name 'Folder 1'
    
    Returns the folder with the name 'Folder 1'
    -------------------------- Example 3 --------------------------
    
    PS C:\> $Parent = Get-UAFolder -Name 'Parent'
    PS C:\> Get-UAFolder -Parent $Parent
    
    Returns all the children folders for the specified folder.
