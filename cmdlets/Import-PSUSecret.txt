
NAME
    Import-PSUSecret
    
SYNOPSIS
    Imports secret from configured secret vaults.
    
    
SYNTAX
    Import-PSUSecret [-AppToken <String>] [-ComputerName <String>] [-Integrated] [-UseDefaultCredentials] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    
DESCRIPTION
    Imports secret from configured secret vaults.
