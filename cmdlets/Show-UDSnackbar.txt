
NAME
    Show-UDSnackbar
    
SYNOPSIS
    Shows a snackbar popup.
    
    
SYNTAX
    Show-UDSnackbar [-Message] <String> [[-Id] <String>] [[-AutoHideDuration] <Int32>] [-Persist] [[-Variant] <String>] [<CommonParameters>]
    
    
DESCRIPTION
    Shows a snackbar popup.
    

PARAMETERS
    -Message <String>
        The message to display.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Id <String>
        The id of the snackbar. If not specified, a new guid will be generated.
        
        Required?                    false
        Position?                    2
        Default value                [Guid]::NewGuid()
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -AutoHideDuration <Int32>
        The number of milliseconds to display the snackbar for. Defaults to 5000.
        
        Required?                    false
        Position?                    3
        Default value                5000
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Persist [<SwitchParameter>]
        Whether the snackbar is persistent.
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Variant <String>
        The variant of the snackbar. Can be default, error, success, warning or info.
        
        Required?                    false
        Position?                    4
        Default value                default
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see
        about_CommonParameters (https://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
OUTPUTS
    
    -------------------------- EXAMPLE 1 --------------------------
    
    PS >New-UDButton -Text 'Say Hello' -OnClick {
    PS >   Show-UDSnackbar -Message 'Hello, World'
    PS > }
    
    Basic Snackbar|Shows a basic snackbar with the message 'Hello World'.
    
    
    
    
    -------------------------- EXAMPLE 2 --------------------------
    
    PS >New-UDButton -Text 'Say Hello' -OnClick {
    PS >   Show-UDSnackbar -Message 'Goodbye, World' -Variant 'error'
    PS > }
    
    Error Snackbar|Shows an error snackbar with the message 'Hello World'.
