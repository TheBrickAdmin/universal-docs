
NAME
    Get-PSUGitStatus
    
SYNOPSIS
    Returns the status of the internal git repo for PSU.
    
    
SYNTAX
    Get-PSUGitStatus [-AppToken <String>] [-ComputerName <String>] [-Integrated] -Commits [-Credential <PSCredential>] [-First <UInt64>] [-HttpProtocolVersion <Version>] [-IncludeTotalCount] [-Skip <UInt64>] [<CommonParameters>]
    
    Get-PSUGitStatus [-AppToken <String>] [-ComputerName <String>] [-Integrated] [-Credential <PSCredential>] -EditInProgress [-First <UInt64>] [-HttpProtocolVersion <Version>] [-IncludeTotalCount] [-Skip <UInt64>] [<CommonParameters>]
    
    Get-PSUGitStatus [-AppToken <String>] [-ComputerName <String>] [-Integrated] [-Credential <PSCredential>] [-First <UInt64>] [-HttpProtocolVersion <Version>] [-IncludeTotalCount] [-Skip <UInt64>] -UncommittedChanges [<CommonParameters>]
    
    
DESCRIPTION
    Returns the status of the internal git repo for PSU. This will return a list of git synchronization jobs.
