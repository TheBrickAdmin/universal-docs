
NAME
    Get-PSUVariable
    
SYNOPSIS
    Returns variables defined in PSU.
    
    
SYNTAX
    Get-PSUVariable [-Id] <Int64> [-AppToken <String>] [-ComputerName <String>] [-Integrated] [-UseDefaultCredentials] [-ValueOnly] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    Get-PSUVariable [-Name] <String> [-AppToken <String>] [-ComputerName <String>] [-Integrated] [-UseDefaultCredentials] [-ValueOnly] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    
DESCRIPTION
    Returns variables defined in PSU.

    PS C:\> Get-UAVariable
