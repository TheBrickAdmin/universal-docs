
NAME
    New-PSUTerminal
    
SYNOPSIS
    Creates a new terminal.
    
    
SYNTAX
    New-PSUTerminal [-AppToken <String>] [-ComputerName <String>] [-Credential <Variable>] [-Description <String>] [-Environment <String>] [-IdleTimeout <Int32>] [-Integrated] -Name <String> [-Role <String[]>] [-UseDefaultCredentials] [-KeepHistory] [-TrustCertificate] [<CommonParameters>]
    
    
DESCRIPTION
    Creates a new terminal. Terminals allow for execution of commands within PowerShell processes in the PSU admin console.
    

PARAMETERS
    -AppToken <String>
        An app token used for authenticating with the PSU management API.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -ComputerName <String>
        The computer name or URL of the PSU management API.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Credential <Variable>
        A credential variable to run the terminal as.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Description <String>
        A description of the terminal.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Environment <String>
        The environment to run the terminal within.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -IdleTimeout <Int32>
        The number of minutes the terminal can remain idle before it is terminated.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Integrated [<SwitchParameter>]
        Executes the command internally rather than using the Management API. Only works when running script from within PowerShell Universal.
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Name <String>
        The name of the terminal.
        
        Required?                    true
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Role <String[]>
        Not supported.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -UseDefaultCredentials [<SwitchParameter>]
        Whether to use the default credentials when connecting to the PSU management API.
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -KeepHistory [<SwitchParameter>]
        Whether to keep the history of all commands and output for terminals.
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -TrustCertificate [<SwitchParameter>]
        Whether the cmdlet should ignore certificate issues.
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see
        about_CommonParameters (https://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    None
        
    
    
OUTPUTS
    System.Object
        
    
    
NOTES
    
    
        
    
    -------------------------- Example 1 --------------------------
    
    PS C:\> New-PSUTerminal -Name 'Terminal1'
    
    Creates a new terminal for the integrated environment.
