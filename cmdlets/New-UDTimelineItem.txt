
NAME

SYNOPSIS
    A timeline item.

DESCRIPTION
    A timeline item.

        Position?                    1
        Default value                {}
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -OppositeContent <ScriptBlock>

        Required?                    false
        Position?                    4
        Default value                grey
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Variant <String>
        The variant of this timeline item. Valid values are filled and outlined.
