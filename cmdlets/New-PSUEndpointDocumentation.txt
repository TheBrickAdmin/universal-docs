
NAME
    New-PSUEndpointDocumentation
    
SYNOPSIS
    Creates a new endpoint documentation definition.
    
    
SYNTAX
    New-PSUEndpointDocumentation [-AppToken <String>] [-Authentication] [-ComputerName <String>] [-Definition <ScriptBlock>] [-Description <String>] [-Integrated] -Name <String> [-Role <String[]>] -Url <String> [-UseDefaultCredentials] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    
DESCRIPTION
    Creates a new endpoint documentation definition.
    

PARAMETERS
    -AppToken <String>
        An app token to access the UA API.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Authentication [<SwitchParameter>]
        Whether authentication is required to access this documentation definition.
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -ComputerName <String>
        The HTTP address of the UA REST API server.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Definition <ScriptBlock>
        Classes defined for input and out types for this endpoint documentation.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Description <String>
        A description of the documentation.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Integrated [<SwitchParameter>]
        Executes the command internally rather than using the Management API. Only works when running script from within PowerShell Universal.
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Name <String>
        Name of this documentation definition.
        
        Required?                    true
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Role <String[]>
        Roles allowed to view this documentation.
        
        Required?                    false

        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false

        The URL used to access this documetation.

        Position?                    named

        Accept pipeline input?       False

        Use default credentials when connecting to the management API

        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Credential <PSCredential>

        Accept wildcard characters?  false
        
    -TrustCertificate [<SwitchParameter>]

        Accept wildcard characters?  false

    None

NOTES
