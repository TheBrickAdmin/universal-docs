
NAME
    Get-UDPage
    
SYNOPSIS
    Get a page for the current dashboard.
    
    
SYNTAX
    Get-UDPage [-AppToken <String>] [-ComputerName <String>] [-Integrated] -Name <String> [-UseDefaultCredentials] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    
DESCRIPTION
    Get a page for the current dashboard. This cmdlet is only valid when running within a dashboard.

        Position?                    named

    -ComputerName <String>

        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Integrated [<SwitchParameter>]

        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Name <String>
        The name of the page to return.
        
        Required?                    true

    -UseDefaultCredentials [<SwitchParameter>]

NOTES
    
    
        
    
    -------------------------- Example 1 --------------------------
    
    PS C:\> Get-UDPage -Name 'Home Page'
    
    Returns the home page. You can pass this to New-UDDashboard -Pages
