
NAME
    Set-PSUComputer
    
SYNOPSIS
    Updates a computer.
    
    
SYNTAX
    Set-PSUComputer [-Computer] <Computer> [-AppToken <String>] [-ComputerName <String>] [-Integrated] [-Maintenance <Boolean>] [-UseDefaultCredentials] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    Set-PSUComputer [-Id] <Int64> [-AppToken <String>] [-ComputerName <String>] [-Integrated] [-Maintenance <Boolean>] [-UseDefaultCredentials] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    
DESCRIPTION
    Updates a computer.
