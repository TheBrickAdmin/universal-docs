
NAME
    Get-PSUJobOutput
    
SYNOPSIS
    Returns stream output for a job.
    
    
SYNTAX
    Get-PSUJobOutput [-AppToken <String>] [-ComputerName <String>] [-Integrated] -Job <Job> [-UseDefaultCredentials] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    Get-PSUJobOutput [-AppToken <String>] [-ComputerName <String>] [-Integrated] -JobId <Int64> [-UseDefaultCredentials] [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    Get-PSUJobOutput [-AppToken <String>] [-ComputerName <String>] [-Integrated] [-UseDefaultCredentials] -RunId <Guid> [-Credential <PSCredential>] [-TrustCertificate] [<CommonParameters>]
    
    
DESCRIPTION
    Returns stream output for a job. This includes error, warning, information, debug and verbose. Use Get-UAJobPipelineOutput to receive pipeline output.

        Accept wildcard characters?  false

        Position?                    named
        Default value                False

        Accept wildcard characters?  false
        
    -Job <Job>
        The job to return output for.

        Position?                    named

        Accept pipeline input?       True (ByValue)

    -JobId <Int64>

        Position?                    named

        Accept wildcard characters?  false

        Use default credentials when connecting to the management API

        Accept pipeline input?       False

INPUTS
    UniversalAutomation.Job
        
    
    
OUTPUTS
    System.Object
        
    
    
NOTES
    
    
        
    
    -------------------------- Example 1 --------------------------
    
    PS C:\> Get-UAJobOutput -JobId 10
    
    Returns output for job 10.
    -------------------------- Example 2 --------------------------
    
    PS C:\> $Job = Get-UAJob -Id 12
    PS C:\> Get-UAJobOutput -Job $Job
    
    Returns output for job 12.
