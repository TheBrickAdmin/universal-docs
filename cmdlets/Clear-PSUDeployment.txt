
NAME
    Clear-PSUDeployment
    
SYNOPSIS
    Clears the pinned deployment.
    
    
SYNTAX
    Clear-PSUDeployment [-AppToken <String>] [-ComputerName <String>] [-Credential <PSCredential>] [-Integrated] [-TrustCertificate] [-UseDefaultCredentials] [<CommonParameters>]
    
    
DESCRIPTION
    Clears the pinned deployment. Performs no action if no deployment is pinned.

        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false

    -Credential <PSCredential>

        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Integrated [<SwitchParameter>]
        For use when running within PowerShell Universal.
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -TrustCertificate [<SwitchParameter>]
        Trust the certificate on the server.
        
        Required?                    false
